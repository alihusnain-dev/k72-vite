# K72 Agency Clone

This project is a high-fidelity clone of the [k72.ca](https://k72.ca/) website, focusing on premium animations, architectural typography, and smooth page transitions.

Made with ❤️ by **Muhammad Ali Husnain** - [@alihusnain-dev](https://github.com/alihusnain-dev)

---

## 🚀 Tech Stack

- **React + Vite**: For a fast, modern development environment.
- **GSAP (GreenSock Animation Platform)**: Powering all the advanced scroll triggers, horizontal marquees, and the "stair" page transitions.
- **Tailwind CSS**: For utility-first styling and responsive layouts.
- **React Router Dom**: For seamless client-side navigation.

## 🌟 Key Features

### 1. **Premium Stair Transitions**
A custom GSAP-powered page transition that resets and triggers on every route change, providing a smooth, high-end feel as you navigate between pages.

### 2. **Hero Sprite Animation**
In the Agency page, the hero image changes dynamically based on the scroll progress, cycling through a team member gallery.

### 3. **Image Stacking & Pinning**
Team sections in "Section 3" feature a sophisticated stacking effect where images pin to the viewport while names and roles scroll past, eventually layered over by the subsequent member.

### 4. **GSAP-Powered Marquees**
Replaced the legacy HTML `<marquee>` with high-performance, infinite GSAP horizontal loops for a smoother architectural vibe.

## 🛠️ Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v16.0 or higher)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/alihusnain-dev/k72-vite.git
   cd k72-vite
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
